## 发现的问题

### 1. API 配置问题 ⚠️
**问题**: 系统尝试连接本地 API 代理 `http://127.0.0.1:15721`，但服务未启动  
**影响**: 无法发送消息和接收 AI 回复  
**建议**: 
- 检查 `.env.local` 中的 `ANTHROPIC_BASE_URL` 配置
- 或启动本地 API 代理服务
- 或配置为直接调用 Anthropic API

### 2. 记忆相似度显示为 0.00 ℹ️
**问题**: 所有记忆的相似度都显示为 0.00  
**影响**: 用户无法判断记忆的相关性  
**可能原因**: 
- 这是一个新会话，还没有用户消息作为查询
- 或者是 fallback 记忆（按时间排序，不是按相似度）
