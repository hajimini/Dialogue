# Claude vs GPT-5.4 对比测试指南

## 快速开始

```bash
# 1. 确保开发服务器在运行
npm run dev

# 2. 在另一个终端运行对比测试
npm run test:compare -- --persona [你的人设ID]
```

## 测试流程

1. **自动运行两次测试**
   - 第一次：使用 Claude (默认模型)
   - 第二次：使用 GPT-5.4 (direct-model-gpt54)

2. **生成对比报告**
   - 报告保存在 `docs/comparisons/comparison-[时间戳].md`
   - 包含每个测试用例的双方回复对比

3. **人工分析**
   - 查看对比报告
   - 根据真实感受判断哪个模型更好
   - 不依赖自动评分，用你自己的判断

## 测试用例说明

测试用例在 `docs/COMPARISON_TEST_CASES.json` 中，包含：

- **COMP_B01-B04**: 情绪回应场景（想哭、很烦、没用、想躲）
- **COMP_A01-A02**: 日常闲聊场景（下班、地铁）
- **COMP_C01-C02**: 记忆衔接场景（猫、展览）
- **COMP_F01**: 敏感话题场景（不想活）
- **COMP_D01**: 角色边界场景（写代码）
- **COMP_E01-E02**: 关系互动场景（戳到我、抱抱）

## 修改测试用例

编辑 `docs/COMPARISON_TEST_CASES.json`：

```json
{
  "id": "COMP_B05",
  "category": "B",
  "category_label": "情绪回应-新场景",
  "input": "你的测试输入",
  "expectation": "你期待的回复特点",
  "prerequisite": null,
  "setup_messages": [],
  "source": "对比测试-情绪回应"
}
```

## 分析维度

查看对比报告时，关注：

1. **情绪回应**
   - 是否先接住情绪？
   - 是否有温度？
   - 是否太理性或太热情？

2. **自然度**
   - 是否像真人朋友？
   - 是否有 AI 味？
   - 语气词使用是否自然？

3. **长度适配**
   - 回复长度是否合适？
   - 是否太简单或太啰嗦？

4. **生活感**
   - 延伸是否自然？
   - 是否刻意"很会聊天"？
   - 是否有生活化的观察？

5. **角色一致性**
   - 是否符合人设？
   - 是否保持边界？

## 下一步

1. 运行对比测试
2. 查看生成的对比报告
3. 用你自己的判断分析哪个模型更好
4. 如果确认某个模型更优，再考虑修改 prompt
5. 修改后重新测试验证

## 注意事项

- 不要依赖自动评分，用真实感受判断
- 关注整体风格，不只是单个回复
- 考虑不同场景下的表现差异
- 注意模型的一致性和稳定性
